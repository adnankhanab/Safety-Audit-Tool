# Safety-Audit-Tool
 Mobile-first, fully client-side web tool for conducting women's safety audits of bus stops and last-mile transport points. No backend, no install, no accounts. Everything runs in the browser; data is stored locally on the device and exported as CSV or JSON.
 What it does — the main features. (Safetipin 0–3 scoring, infrastructure checklist, GPS, photos, CSV/JSON export.)
How to use it — open the link on a phone, grant location/camera permission, score each spot, save, export.
